## 🛠️ 技术实现

### 鼠标事件处理
- `mousedown` - 记录按下状态和初始位置
- `mousemove` - 计算鼠标偏移并更新摄像头
- `mouseup` - 释放状态标志
- `wheel` - 处理滚轮缩放（已禁用默认滚动）
- `contextmenu` - 禁用右键菜单

### 摄像头控制
- 使用 Cesium 的 Camera API
- 支持摄像头方向（heading）、仰角（pitch）、滚转（roll）
- 约束仰角在 -90° 到 90° 范围内

### 灵敏度配置
所有交互的灵敏度都可配置：
- 旋转灵敏度：`0.005`（在 `rotateModel` 函数）
- 移动灵敏度：`0.001`（在 `panModel` 函数）
- 缩放灵敏度：`0.1`（在 `zoomModel` 函数）
